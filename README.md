# Mantenimiento de Inventario

## Descripción del Proyecto

Este proyecto es una aplicación web simple y intuitiva para el mantenimiento y gestión de inventarios. Diseñada para facilitar el control de stock de productos, permite a los usuarios agregar nuevos productos, revisar el inventario existente, generar reportes básicos y monitorear estadísticas clave como el total de productos y aquellos con stock bajo.

La aplicación está construida utilizando tecnologías web estándar (HTML, CSS y JavaScript) para una experiencia ligera y accesible directamente desde el navegador, sin necesidad de instalación adicional.

## Características Principales

- **Búsqueda de Productos**: Campo de búsqueda para encontrar productos por nombre, categoría o código.
- **Agregar Productos**: Botón para acceder a un formulario de adición de nuevos productos (funcionalidad simulada en la versión actual).
- **Ver Inventario**: Opción para visualizar el listado completo de productos en el inventario.
- **Generar Reportes**: Herramienta para crear reportes del estado actual del inventario.
- **Estadísticas en Tiempo Real**: Muestra el total de productos, cantidad de productos con stock bajo y la fecha de última actualización.

## Tecnologías Utilizadas

- **HTML5**: Estructura de la página web.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Lógica básica para la interacción y simulación de datos.

## Instalación y Uso

1. Clona o descarga el repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web preferido.
3. La aplicación se cargará automáticamente y estará lista para usar.

No se requieren dependencias externas ni servidores; es una aplicación cliente-side pura.
